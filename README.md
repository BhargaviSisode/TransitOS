# TransitOS – AI Urban Mobility Control Center

TransitOS is a smart dashboard for monitoring and optimizing public transportation systems in real time.

## Features
- Fleet monitoring
- Demand prediction
- AI route optimization
- Transit analytics
- Map visualization

## Tech Stack
React, TypeScript, Node.js, Gemini AI

## Run Locally
npm install  
npm run dev

## Hackathon Submission

This project was developed as a prototype for a hackathon challenge focused on intelligent urban mobility systems. The goal is to demonstrate how AI-powered analytics and real-time monitoring can improve public transportation efficiency.

The system integrates fleet monitoring, demand prediction, analytics dashboards, and AI-driven route optimization to assist city authorities in making smarter transit decisions.
